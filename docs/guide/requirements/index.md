# Requirements

RGS uses multiple technologies. We use [Devbox](https://www.jetify.com/devbox) to manage development dependencies.

**Setup:**
1. Install [Devbox](https://www.jetify.com/devbox/docs/installing_devbox/)
2. Navigate to the RGS directory
3. Run `devbox shell` to install dependencies (see `devbox.json`)

**Additional requirements:**
- [Development Environment](development-environment.md)
- [Docker](docker.md)
- [Bun](bun.md)
- [Rust](rust.md)
